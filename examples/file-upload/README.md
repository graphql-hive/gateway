## How to run locally?

1. Install
   ```sh
   npm i
   ```
1. Start service bucket
   ```sh
   npm run service:bucket
   ```
1. Compose
   ```sh
   npm run compose
   ```
1. Start the gateway
   ```sh
   npm run gateway
   ```
