   ```sh
   npm i
   ```
1. Start service authors
   ```sh
   npm run service:authors
   ```
1. Start service books
   ```sh
   npm run service:books
   ```
1. Compose
   ```sh
   npm run compose
   ```
1. Start the gateway
   ```sh
   npm run gateway
   ```
